バトオペ出撃Notifier
====

Overview

## Description
PS3版 機動戦士ガンダム バトルオペレーション用
出撃エネルギータイマーカウントアプリです。

各テキストをタップすると値を編集可能です。
120分経過すると支給エネルギーが加算されます。
その際、バイブ＋ステータスバーにてお知らせ。
