バトオペ出撃Notifier
====

PS3版 機動戦士ガンダム バトルオペレーション用、出撃エネルギータイマーカウントアプリです。

## Description
各テキストをタップすると値を編集可能です。
120分経過すると支給エネルギーが加算されます。
その際、バイブ＋ステータスバーにてお知らせ。
